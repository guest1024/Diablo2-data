# 中文来源补充建议（执行版）

## 结论

当前更值得优先补的中文高质量来源：

### 第一优先级
1. `91d2.cn`
2. `ttbn.cn`

### 第二优先级
3. `diablo2.com.cn / bbs.diablo2.com.cn`

## 选择依据

### 91D2
更偏中文资料站/教程站，当前已检索到的高价值内容包括：

- 新人冰法 MF 指南
- 角色职业天梯 Build 练法
- 85 级场景攻略
- 插件/地图常见问题

适合作为：

- 中文 FAQ
- 中文 build / 开荒 / MF 指南
- 中文术语与经验层

### TTBN
更偏中文资料库 / 资料聚合站，当前已检索到的高价值内容包括：

- 装备大全
- 场景地图
- 掉落 / TC 相关资料
- FAQ / 雇佣兵 / 常见问题
- 热门攻略

适合作为：

- 中文资料库
- 掉落/地图/装备补强层

### CNBN / diablo2.com.cn
有价值，但当前环境访问不稳定。更适合作为：

- 中文社区补强层
- FAQ / 经验 / alias 俗称层

不建议目前作为自动抓取主来源。

## 建议抓取方向

### 91D2
- 新手指引
- 角色职业
- 场景地图

## 并入策略

中文来源优先并入：

- `merged/chunks.jsonl`
- `merged/aliases.jsonl`
