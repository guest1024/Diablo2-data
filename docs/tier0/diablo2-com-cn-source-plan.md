# diablo2.com.cn / CNBN 中文社区补充计划

## 价值判断

该站点对当前项目的价值在于：

- 中文术语
- 中文 FAQ / 常见问题
- 中文 build / 配装 / 角色经验
- 1.09 / 1.13 社区经验
- 中文玩家常用俗称 / 缩写 / 说法

它适合作为：

- 中文补强层
- alias / FAQ / build / 经验层

不适合作为：

- 最高权威事实源
- 交易 / 灌水 / 活动类主语料源

## 推荐抓取板块

优先级从高到低：

1. `暗黑2资料分享`
2. `暗黑2角色讨论区`
3. `PC软硬件配置区`
4. `暗黑1.09战网讨论区`
5. `暗黑1.13战网讨论区`

## 明确不抓

- 交易区
- 估价区
- 拍卖区
- 灌水区
- 文化杂谈
- 战队管理
- 活动区（除非有高价值 FAQ / 规则总结）

## 帖子筛选规则

只保留：

- 置顶帖
- 精华帖
- FAQ / 教程 / 索引帖
- 结构化长文
- 职业 build / 路线 / 技术说明
- 游戏运行 / 报错 / 环境兼容说明

直接排除：

- 纯交易
- 纯估价
- 单行求助
- 水贴
- 无正文短帖

## 当前抓取结论

通过搜索结果和站点摘要，已确认该站点存在以下高价值板块：

- `暗黑2角色讨论区`
- `暗黑2资料分享`
- `PC软硬件配置区`

但在当前执行环境里，对 `bbs.diablo2.com.cn` 的直接 HTTP/HTTPS 访问出现超时/不可稳定抓取问题，因此暂不做自动化全量正文抓取。

## 后续补充方式

优先顺序：

1. 等网络可达后，按版块白名单抓取置顶/精华帖
2. 只提取正文 + 标题 + 语言 + source 元数据
3. 进入中文 alias / FAQ / build 经验层
4. 不直接并为最高 authority canonical fact

## 与现有 merged 主库的关系

该站补进来后，建议优先进入：

- `merged/chunks.jsonl`
- `merged/aliases.jsonl`
- `merged/provenance.jsonl`

而不是优先直接进入：

- `merged/canonical-claims.jsonl`

除非内容是明确、稳定、可交叉验证的事实知识。
