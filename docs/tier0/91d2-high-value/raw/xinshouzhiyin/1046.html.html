新人冰法mfsor打宝mf指南
打宝，简称mf，通过杀怪获取有价值的魔法装备，是暗黑2游戏的重要内容。冰法由于具有许多优势，如伤害高，能瞬移和易上手等，配合装备和佣兵，成为mf的首选角色。为了新手的快速入门和提高mf的效率，分享个人的经验是有益处的，希望大家指正。
1、人物：
（1）属性：力量加到156（含装备）；敏捷不加或够穿装备；生命加满；法力不加（或加50点左右）。
（2）技能：冰系：暴风雪及相关技能加满，冰支配加满，冰封装甲加1级就够；火系：温暖加1级；电系：瞬移和静电力场各加一级。
（3）等级：85级以上，以提供充足的属性和技能点数。
（4）状态：mf值300+，暴风雪伤害3000+，生命1000左右，快速施法速度fcr达到105%以上，抗性尽量撑到正值。
2、装备：
（1）人物：头盔（暗金军帽、塔因头盔、三孔头镶黄宝石），衣服（暗金罗瑟战甲、四孔甲镶黄宝石），武器（暗金涡流水晶、精神剑），盾牌（精神统盾、押韵），腰带（暗金重扣带），手套（暗金锁链手套），鞋子（暗金战场之靴、mf鞋），戒指（拿各），项链（马拉或mf项链）。其他加mf值装备有塔拉夏项链，衣服和腰带三件套，坦克雷鞋子和项链等。包裹里放基德的运气，7mf护身符，加生命抗性技能护身符若干。
（2）佣兵：吸血鬼凝视或塔拉夏头，都壳或物免甲，眼光（精英底材）。
3、路线：
   地狱难度下mf的常见场景有安达利尔，督瑞尔和墨菲斯托等boss，崔凡克议会成员，暴躁外皮，85级场景等。由于bugmf机制的存在，即boss首杀掉落，bugmf时装备掉落成色最低为蓝色，并且不掉落药水，箭矢等物品，为mf提供极大的加成。新人建议mf各关底boss，安全，高效且操作简便。
   *实现bug的方法通常是在人物初次过关时，利用已通关的角色代替自己杀boss，人物不在现场而形成bug。
   *判断是否为bug的方法是游戏中按Q键打开任务状态栏，看右上角是否有红色❌，如果有，表示非bug。
   新人推荐mf督瑞尔，也叫虫子，先从传送点到第二幕术士的峡谷，然后进入塔拉夏的古墓中虫子的入口，由于入口狭小且经常聚集怪物，建议冰法用心灵传动技能入洞。正常一次mf时间在30秒左右，两个小时mf大约200次，运气好的话就可以收获1-2件有交易价值的装备啦。
4、目标：
   俗话说“冰法养全家”，游戏中有交易价值的装备挺多，不过掉率都非常低。按照官方物品的掉率，有交易价值的装备掉率一般在1/1000左右，而一些非常罕见的装备，如格里风之眼，掉率甚至低于1/10000，纯属小概率事件，极度依赖人品。^^
   通常mf的目标定义为实用性强，通用性强，需求量大的暗金或绿色装备，如军帽，蛇皮，乔丹等，可以参考交易区的物价表。
   在mf冰法装备成型的情况下，通常mf2-3个小时，就可以收获1-2件有交易价值的装备啦。
   暗黑2人人都有一个得心应手的mf冰法。mf需要毅力和坚持，因为神器出现的那一刻太让人激动了！
