资料片一帖通（任务tips、小号PASS、牛场防破处、BUG击杀、元素BUG）
该帖用来帮助资料片中的小伙伴们更好地完成任务，尤其是带小号时。
本贴内容只涉及任务知识点，RR流程还请参阅各大精华帖。

小号升级：
01-10 ACT1野外
10-20 ACT2野外/古墓
20-25 普通KC
25-40 普通KB
40-85 噩梦KB+地狱KB


ACT1

1.邪恶洞穴
先用大号进去把怪物清理到剩下最后一只，然后开门回城，小号进去击杀之即可完成。

2.修女埋骨之地
8开带小号时，首先带小号完成该任务，7个小号都获得一名ACT1-PET，之后用强化法师给每个PET都加上高级强化，这样每个小号们都带着一位妹子出门开火车，一路自动乱射升级很快，此方法适合直接在资料片RR的小伙伴使用，毕竟不能直接进地狱牛场蹭级。

3.凯恩的搜寻
RR之后的小号有兴趣的可以完成该任务，可批量生产黄金戒指，物品等级60，除力量、生命和双吸以外的大部分属性可以满档，尤其是抗性和法力。

4.遗忘之塔
用未完成任务的角色建立游戏，再用已完成任务的角色击杀女伯爵，任务箱子就会喷钱和物品，虽然这个箱子不会掉钥匙，但是能增加一些掉落物，如果想要保留这种额外掉落能力，就不要让未完成任务的角色参与击杀现场而导致任务完成。

5.交易的工具
完成该任务最起码也要达到8级才行。

6.屠杀的修女
完成任务后请务必直接与瓦瑞夫对话去ACT2，此后安达利尔自带BUG击杀。

7.牛场破处
所谓牛场破处就是无法再开启牛场红门了，有两种情况：
（1）已PASS当前难度巴尔任务的角色击杀了母牛之王。
（2）角色无论是否PASS当前难度巴尔任务，只要组队的队友由于击杀母牛之王破除了，也会跟着被破。
解决办法：开门不进门，进门不PASS，进门可破处。
开门不进门，PASS巴尔的角色拥有开门能力的同时也有被破除的危险，只要永远不进牛场就永远能开门。
进门不PASS，进去干活儿的如果是尚未PASS巴尔的角色，击杀母牛之王也不会破除。
进门可破处，进去干活儿的如果是已经PASS巴尔的角色，先退组，然后直接把处破了，以后也随便杀。


ACT2

1.杰海因消失之谜 与 bus使用注意事项
1.1 水井KEY
（1.1.1）水井KEY建立游戏后，bus不要在ACT2下线的状态下直接进入游戏，这样一进游戏bus就处于ACT2场景内（水井边）且靠近王宫范围，此时只要bus一出城，就会导致杰海因立即消失（水井与王宫两边都不在）；
（1.1.2）只要bus不靠近王宫触发范围就不会使杰海因消失，比如与ACT3马席夫对话回到ACT2，此时直接再从马席夫去ACT3，杰海因并不会消失；注意：ACT2小站、城门、下水道、王宫入口、回程卷传送门都可触发。
（1.1.3）水井旁边的杰海因需要小号守一下，原因是当击杀督瑞尔事件触发时，城里在没人的情况下杰海因会随游戏房间的任务状态更新至王宫处；当BUS击杀督瑞尔时，小号在水井边上看着杰海因，他的进度就被小号控住了，此时BUS击杀督瑞尔，杰海因就不会去王宫了。
（1.1.4）bus在与泰瑞尔对话后，要依次回城与杰海因和马席夫对话，重置bus自身的任务链状态，不然下次游戏时bus再与泰瑞尔对话将无法触发任务链，导致小号无法完成任务。
（1.1.5）未完成插杖任务的新角色建立的游戏，当bus靠近王宫或从水井边进入ACT2也会触发杰海因任务链状态刷新，在bus出城后杰海因会立即从水井消失而出现在王宫门前。
一.玩家人物无论有没有任务进度，只要进入ACT2绝大部分区域，一出城就会使杰海因从水井边回到王宫，而水井KEY进度跳过了主线，杰海因既不能在王宫，又已离开水井，所以消失了。
二.杰海因消失了没关系，只是暂时的，重新开一场游戏他就又回来了。
1.2 正常KEY
与水井KEY的主要区别就是，正常KEY的主线任务链触发状态与bus进度一致，不会导致杰海因位置冲突而消失，他始终都会在王宫等待任务交付，bus同样需要重置自身的任务链状态，正常KEY建立的游戏可从王宫下去开门快速穿梭ACT2城内区域。
1.3 水井KEY失效
水井KEY失效是由于KEY号与NPC有了交互，导致任务链被触发，虽然合成杖与击杀召唤者的任务都没完成，但是主线任务插杖已经完成了，所以杰海因的任务进度被强制更新到了王宫，以后该KEY号再建立的游戏，杰海因不会再消失了，因为他会直接从王宫出现，等于“水井”失效了。没有任何补救办法，只能重新做一个水井KEY。
一.水井KEY任何时候都不要与杰海因对话
二.水井KEY建立游戏，等其他人物进入游戏后，水井KEY就立即退出，永远不会“被失效”

2.七个古墓
BUG督瑞尔的制作方法，未PASS的角色全部等在门外，等待已PASS的角色击杀督瑞尔，然后指定一名未PASS的角色再进门与大天使对话即可完成BUG，之后回城依次与杰海因和马席夫对话进入ACT3。


ACT3

1.黄金鸟
用未完成任务的角色从城镇出门一路往前走，遇到的第一个精英/头目怪物就会掉落黄金鸟。

2.古代宗教之刃
吉得宾与凯恩任务一样，推荐RR小号们完成，可批量生产黄金戒指，物品等级75，除了双吸以外的所有属性都可以满档。

3.议会
带PASS时直接躺一个小号在崔凡客的3C门口，然后大号击杀全部精英级3C即可完成，不用合成任务道具，此处还顺便掉落赫拉迪克方块。

4.守护者
BUG墨菲斯托的角色注意不要PASS该任务，使用已PASS的角色建立游戏，再用未PASS的角色击杀墨菲斯托就是BUG击杀。


ACT4

BUG大菠萝的角色注意不要PASS该任务，BUG击杀同墨菲斯托。


ACT5

1.亚瑞特山的救援行动
救野蛮人的任务，先用大号站远一点把3座门用远程技能全部砸开，挑一个门把门口稍微清理一下，然后开门，小号出门后直接跑向门口即可完成。
如果BUS不小心触发了一个也没事，去下一个再开传送门，只要小号靠近时头顶出现“跟我走”即可，虽然小号任务面板会显示无法完成，其实已经完成了，重进游戏会看到进度已经到与NPC对话。

2.哈洛加斯的反叛行动
击杀尼拉塞克与痛苦之厅小站不可同时完成，否则安雅旁边的红门就会消失。

3.通道的祭典
3个野蛮人先祖可以使用元素BUG，大幅降低击杀难度，使用元素BUG的角色站在最左下角即可，对话后只有最左边的旋风野蛮人会动。

4.毁灭的前夕
巴尔也可以使用元素BUG，使用元素BUG的角色站在王座不进世界之石即可，巴尔只会原地扭动。
BUG巴尔的角色注意不要PASS该任务，BUG击杀同墨菲斯托。

8PP全手动BUGKB流程：
打完5小队，法师找准位置开门，元素BUG角色与其他不进大殿角色不站在传送门附近，狂热PLD、审判PLD、标马，这3PP站位距离门5码以内；
法师进入大殿，TP至巴尔身边，SF至50%以下，使用指令2指挥3PP回城，开启传送门，使用指令1召唤3PP进入大殿传送门；
3PP入场后
1.法师传送至桥外侧，由MF标马进行击杀。
2.标马血量压到1~2%以内后跑向桥外侧或回城，由MF法师进行击杀。

注意事项：
1.一定要有一名元素BUG角色把巴尔定住，不然极大影响标马输出。
2.进大殿角色中必须有高级狂热PLD和25级审判PLD两位工具人，提供给标马足够的伤害增益，且都不带PET。
3.标马和法师，选择其中一人穿全套MF即可，另外一人在最后阶段注意远离巴尔，以免PET抢杀，不带PET的角色可不动，BUS法师一般都要带，专职打手的法师和标马可不带。
4.如果队伍里有召唤系NEC控场，其余角色不带PET，5小队手动输出靠标马和法师足以，这样除了元素BUG角色依然留在王座，其余6PP均可随指令空降，蹭点经验也是好的，如果带PET则不进大殿以免PET抢杀，或者用大殿红门进场蹭经验。

击杀时间：
大殿TP+SF大约3秒，开门+指令来回大约3秒；
如果用MF标马击杀巴尔，法师SF至50%以下后，剩余血量全身MF装标马可在10秒内完成击杀。
如果用全套MF标马+全套MF法师，则两者的PET都可参与击杀，两名MF角色可同时攻击，剩余血量可在5秒内完成击杀，由于两名角色的MF值都相当高，谁击杀最后一击都可以。

另外，不要等PET击杀巴尔，佣兵对关底BOSS只有25%伤害，且还要计算命中、格挡，不差那点MF。

尼拉塞克速杀指南：
尼拉塞克抗性，冰70 火电33，天生不免疫三元素，抗性免疫主要看随机词缀。
纯单系电火冰皆可，关键要有无限和大BO，格挡型为最佳。
战术高度统一，全部TP到脸上狠怼即可，冰风暴、大火球、闪电，打得他连续HR放不出尸爆，以3秒内怼脸击杀为及格。

尼拉塞克祭坛内有BUG蛇就N，祭坛外围有蛇没关系，速度怼完捡东西走人。

VIT和格挡的争议由来已久，玩家们几乎都是在讨论两者的利弊，可惜都不在一个频道上说话，因为几乎没人关心这个问题的本质是由什么导致的。

格挡与不格挡是由玩家的操作形成的一种符合个人风格的选择，有人喜欢冲锋陷阵正面硬钢，有人喜欢操作细腻完美走位。
扛着怪打的肯定会承受更多攻击，显然格挡更高效的减伤能带给操作者明显更“硬”的体感，他们会说：“还是得格挡，不然生存无法保证”；
避着怪打的肯定会更少受到攻击，显然防御和格挡都没有能发挥的足够空间，他们会说：“卵用没有，要格挡干嘛？浪费点数还影响施法”；

玩家们往往很难从自身开始找原因，而总是先偏向于找外部因素，所以这两个群体之间没有任何一方能说服另一方，因为他们说的话根本就不是建立在一个共同的基础之上。
