二、准备工作
建立必备人物7个，分别是bus，act2 key 3个，em act1红门key，dy act1红门key，专职波霸1个。
1、多窗口准备：
建9个游戏的快捷方式，起名字1-7，bus，key，属性里添加 -title 1-7 bus，key，这样窗口不会在任务栏重叠
2、win7快速选择窗口：很多童靴都是XP，很悲哀告诉你，XP不行
alt和ctr中间的windows键+数字键可以快速选择窗口，没必要用鼠标点和alt+tab，精准定位窗口。
3、快捷方式参数，都用上，
-w 切换到窗口模式
-ns 无声模式
-skiptobnet 直接进入battle.net
-nonotify关闭错误信息报警
-title  设置窗口标题为
-nohide 窗口不允许最小化
-lq 降低画质，提高cpu利用率
-res800 -res800 窗口大小为800x600
-Pdir RR加载RR地图


三、水井KEY制作流程
制作水井KEY：
普通难度：建A,B,C,D四个BB，一个BUS，
    A建游戏，B,C,D,BUS进，杀安姐，A留在ACT1，B,C,D,BUS去ACT2，BUS去绿洲虫子洞开门，D进去拿杖，BUS去蛇殿神坛开门，D进去拿XL，然后杀召唤者，到古墓插杖的地方，D用BUS的箱子合成，A通过对话 来到ACT2（原地别动），D插杖，A不和任何NPC对话走到小站回ACT1然后退出游戏（普通的KEY就做好了）。以此类推，B和C分别做噩梦和地狱的KEY。（D就没用了）
水井key的使用：（普通、噩梦、地狱都是如此）
    7BB过A1退出（不要去A2），BUS也退出，KEY建游戏，BUS进，KEY退出，BUS清理ACT4，ACT3等等，在去ACT2之前，7BB中的一个进游戏，通过对话到ACT2，然后杰海因就会在水井边朝BB走过来，BB和杰海因对话（鼠标点一下并且不要结束对话），BUS去ACT2术士峡谷（开个门），进古墓虫子洞杀ACT2BOSS，飞到天使眼前（看着天使不要对话），另外6BB进游戏，分别对话来到ACT2，BUS确保7BB都到了ACT2之后和天使对话，7BB分别和杰海因对话就可以去马媳妇那里过关了，BUS也通过天使开的门回城重置BUG（和杰海因对话，和马媳妇对话）。


四、RR详细流程（做5个戒指任务）
【PT难度】
1：7BB建房进房，进BUS，BUS飞到安姐身边开门，进1BB，KA回城，全退。
2：PTKEY建房，进BUS，KEY退出。
3：BUS飞去剥皮森林拿刀，回城扔奥玛斯身边。
4：BB1进游戏水井旁和杰海因对话，不要关闭对话。
5：BUS到ACT2术士峡谷开门。
6：BUS去ACT2 K虫，K完后到大天使身边。
7：进7BB（进到5BB的时候先和瓦瑞夫说话去ACT2，再进6和7，防卡CHAR），集体到ACT2。
8：BUS和天使说话回城，通过天使开的门回城。
9：7BB依次和杰海因说话，飘逸去和马夫说话去ACT3。
10：7BB漂移到奥玛斯处，BB1拿刀和奥玛斯说话两次。
11：BUS和杰海因说话，再返回和马夫说话，通过自己在术士峡谷开的门去ACT3（KEY重置）。
12：BUS飞3C处找安全地开门，BB1进，BUS K3C，和BB1同回城。
13：BB1和凯恩说话，BUS飞去老墨处在河中间平台处开门（注意贴边飞别引来右下方的瘸子）。
14：BB1进门，BUS杀老墨。
15：BB1去ACT4，6BB依次和奥玛斯说话拿RING，和凯恩说话进门去ACT4。
16：BUS去敲石头处清场，BB1进门敲石头。
17：ACT4开封门KD完成流程
【噩梦难度】
流程同PT。这里只列出不同地方：
ACT1:
1、EM红门KEY建游戏，BUS进游戏。
2、EM红门KEY退出游戏，职业波霸进。
3、BUS任意传送点开门，职业波霸bobo后退出，ALL BB进游戏。
4、BUS石矿荒野进红门，凯恩处开门，BB1进救凯恩。
ACT2:
1、EM ACT2 KEY建游戏，BUS进游戏。
2. EM ACT2 KEY退出游戏，职业波霸进。
3、BUS任意传送点开门，职业波霸bobo后退出，ALL BB进游戏

【地狱难度】
流程同EM，DY ACT2 KEY建房后，给BUS bobo。BUS拿刀后分别到3C和老墨处见面锁血(1PP血量)。
要点：冰免先sf到丝血，换新星打死。记得开门以防万一，好领尸体。


五．BUS加点
50级技能加点，合计61技能点！
1、冰系30点：冰封球max、冰支配5；冰护甲不加，点数太紧张
2、电系30点：tp9，静电力场16，新星3，电支配1
3、火系1点：温暖1
升级后加点：
1、优先max冰支配。
2、max冰蛋加成冰封球。可以考虑+1点冰甲。
3、冰系加完已经80+级别，fz练级很难的，RR来说不用考虑升级了。

50级属性加点
1、力量 够装备
2、敏捷 够装备
3、精力 开荒加到80-100， 如果还是觉得不够用，加到120
4、生命 ALL

六、常见问题
问题一：RR需要多少时间
拿完5戒指新手总体耗时大约30分钟，未达标的多练几次，熟能生巧。
熟练工：17分钟算合格，13分钟算优秀，10分钟以内叫爸爸！

问题二：为什么不加暖气和TP
加这两是为了节省蓝，蓝的解决方案：
1、属性点加精力到80-100。一般来说不bo蓝量450刚刚够用，550以上绰绰有余。这样省去你购买10% 高mana 抗性戒指，例如：10% 50mana 25fr=6ist（开荒前期物价）
2、地图不要屏蔽蓝瓶，随捡随喝。

问题三：是否那任务戒指
1、单日RR次数总数，如果只有1~6次，那肯定那。
2、如果RR次数总数8+次，建议一个也别拿。

问题四：血量多少合适：不bo 600+够用，700+足够，800+安逸。继续增加的收益断崖式下跌，RR需要的装备门槛较低。
