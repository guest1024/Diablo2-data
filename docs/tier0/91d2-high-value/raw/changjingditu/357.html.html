暗黑HC专家模式FZ MF热刺专用对象应该是PT冰冷乌鸦

原理：

   冰冷乌鸦是7级的固定金怪（随从也是7级），而热刺最低等级也是7级，因此她能掉的东西的种类很少，只有皮靴、匕首等低等级装备，所以相对而言出热刺的几率就大多了（也不会出等级高的绿皮靴）。
   因此用穿MF装的角色杀PT A1第一个小站，洞窟第一层的冰冷乌鸦，可以获得最高掉落几率的热刺（还有海鸥）。可以用以下的法师，也可以用法师TP再用MF BAR杀之（主攻最好用战斗狂嗥），还可以把SOR堆高MF而不管FCR……。这几种都各有优缺点，第一种速度快而MF值低，第二种MF值高而速度慢，第三种介于前两者之间。故而以下均为参考而已，有兴趣的可以自己发挥，不必局限于此。
 
技能：
   传送、温暖加满，可使TP时法力充足，主攻技能选连环闪电或冰封球，加几点即可。
 
属性：
   力量STR： 到35凑够金色包袱要求的45。
   敏捷DEX： 不加。
   活力VIT： 剩余全加在这，少些也不用太在意，7级及以下的怪不容易弄死人的。
   精力MANA：视情况可以加到100。
 
原则：
   在流畅的前提下，尽量提高MF值，而基本不用考虑防御和攻击力。
 
装备：
   法拳和无眼之墙，凑够FCR 40，因为可以忽略盾牌的防御作用，所以可以选择用盾牌来凑FCR。
   海鸥、3孔镶黄头、双拿各、金色包袱、坦克雷项链和靴子，凑MF值（100+72+60+30+78=340）。
   抽动凑力量。
 
方法：
   用穿MF装的法师杀PT A1第一个小站，洞窟第一层的冰冷乌鸦，获得最高掉落几率的热刺，完全不需要BO BAR。
 
说明：
   1、冰冷乌鸦每次都会掉6件东西（与房间内的人数无关），但大多数情况下有4件是药瓶等垃圾，一般只有两件是装备或碎裂的宝石。
   2、洞窟第一层可以修改地图文件，以便能标出方向。进了洞窟第一层以后，她就在离第二层最近的那个大分支里（不是拐角的地方）。
可在文件 d2hackmap.cfg 里相应位置加上以下一行即可。
Minimap Cells[3][9]:     1,0x20　 //冰冷之源-洞窟第一层，MF热刺用
   3、每次MF一般只需要不到20秒，所以经常会出现MF完退出后需要等一会才能建新游戏的情况。比较好的解决办法是：练两个角色（不在同一ACC里），替换着MF。
   4、因为是PT A1 MF，因此角色等级可以低些，三、四十级也够用了。
   5、沿路地表上的妖怪都是低于7级的，不用去管，洞窟里也有只有黑色流浪者（和冰冷乌鸦一样）金怪及随从才是7级的，可顺便MF之，其余的都是低于7级的。
   6、也可以雇用一个弓箭MM，帮你收拾下可能漏下的怪，顺便给你当保镖，只要不去其他场景，她就会一直跟着你。
   7、可以升到高级别后 BUG KM，加满静力场，用冰球最后一击。
   8、随身带上几颗无瑕疵的骷髅，因为偶尔路上会遇到升级宝石的树，40个非资完美骷髅也值24#呢。
