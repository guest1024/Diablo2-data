请教一下pp数和nodrop的问题
问题：
常在BBS看到大家建议1pp刷古代通道的金怪，这个和资料室的上述内容一直，即金怪的不掉率为0，无需增加pp数。又常看到建议至少3pp刷老墨等BOSS，确保掉6件。那么不是和金怪不掉率为0矛盾吗？请大家解惑谢谢！
回答：
关底boss都有空档，所以pp数仍旧提高掉率
固名金怪有的有空档，有的没有。3c有空档，皮叔没有。一般任务金怪是有空档的

随机金怪都没有空档
蓝怪原则上没有空档，但是有60%几率掉钱不掉装备，五小队也属于蓝怪
