超级暗黑破坏神(Diablo Clone) (恶魔)
除了少数精英玩家, 对大部分玩家来说, 超级暗黑破坏神是个非常神秘的恶魔. 注意, 超级暗黑破坏神比普通的暗黑破坏神强壮太多了! 但是等待着那些充满智慧和力量的玩家勇敢的迎接挑战的回报是非常丰厚的...
超级暗黑破坏神只会在战网国度中才会被唤醒, 那什么才能唤起他呢? 对一些人来说这是一个谜, 另一些人却知道. 这和乔丹之石有关吗? 超级暗黑破坏神只会出现在地狱难度, 他可以出现在任何场景的任何地方, 它是可以掉落毁灭(暗金小型护身符)的唯一怪物.


提示及其他信息
在于暗黑破坏神战斗时尽量保持最高的火焰和闪电抗性. 以及一些物理免疫和闪电/火焰吸收属性, 这样可以利用他的攻击伤害来为你治疗. 
暗黑破坏神会对你, 你的雇佣兵, 城镇传送门施放骨牢. 所以在使用传送门或死亡后回来拿尸体时一定要小心. 
红色的巨型闪电会造成一般的闪电伤害和一半的物理伤害. 尽量高的闪电抗性和伤害减少/高格挡率属性可以让你在这种攻击下尽量安全一些.
这与你之前碰到的暗黑破坏神最大的不同在于有非常高的生命, 非常高的抗性, 更加强力的攻击, 和技能 毁天灭地. 他的生命恢复速度很快，保重!
超级暗黑破坏神使用的技能包括: 火焰新星, 巨型红色闪电, 寒冰掌, 火墙, 烈焰风暴, 骨牢, 突击, 嘲弄, 毁天灭地。
