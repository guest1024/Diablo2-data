实测ACT5古代野蛮人3BB的火焰强化bug

准备工作
1. 修改暗金怪物的随机赋予属性仅仅有火焰强化，其他属性几率设为0
2. 3bb的基础生命值修改为10000，空手伤害修改为0，临界一击百分比修改为0. 
3. Flying Axe(马道克的武器)和Colossus Voulge(科力克的武器)的伤害修改为固定100.
4. 所有词缀的属性修改为空。目的是使得几位bb的武器不会附加额外的伤害。
5. 修改一个测试存档。火焰抗性上限修改为100. 并拥有三种元素伤害手段来对付三位bb。存档通过换装备可以有0DR/100FR、0DR/90FR、50DR/100FR等不同状态，通过切换正副手，还有30000防御&75%格挡和200防御0格挡的区别。

测试一
单人难度。100FR/50DR状态。杀掉其他两位bb，只留马道克一位(飞斧投掷). 副手无盾状态接受攻击数十次，每次伤害均为100，并无额外伤害。符合攻击伤害理论值。
此测试说明，火焰强化属性bb的攻击伤害正常，并无额外伤害。

测试二
单人难度。100FR/50DR状态。杀掉其他两位bb，只留科力克一位(跳跃攻击). 正手高防御75%格挡状态接受攻击数十次。科力克频繁击空，但生命值在减少，减少时bb身上伴有火焰伤害击中的那种效果，而此时并非科力克发起攻击的时候，而是固定的一个频率，个人的感觉为2秒一次。而且，bb身上出现火焰效果时并不一定会减少生命。
此测试同测试一作比较，说明，火焰强化bb在近身时会有固定频率的脉冲伤害，而并非攻击时附加额外伤害，也并非在被打出HR打击恢复状态时出现伤害。同时也说明，此伤害可以被格挡。
科力克自身普通攻击伤害为100，vs 50DR时为50. 自身基础生命值为10000，按照公式计算，死后尸体爆炸后的伤害为10000*20%*50%=1000(单机公式，战网要除以4). 60-100%的随机伤害，一半物理一半火焰。100FR/50DR状态时，实际伤害为150-250

可以看出，同测试二相比，额外伤害数值均超出了尸爆伤害的范围，而测试二中，也有少量超出范围的伤害。这些超出范围的伤害，大致都在尸爆伤害的二倍。对比测试二跟测试三，唯一的区别就是格挡。那么可以得出结论，额外伤害每次作用两次。而在测试二中，两次连续伤害都没有被格挡掉的几率很小，所以伤害翻倍很少出现。(注：事先已经屏蔽掉了临界一击)

测试四
输入命令/players 8，重复测试三的测试。数据不再给出，结论是：人数并不影响这个额外伤害的数值。

测试五
将3bb的基础生命值修改为100000，结果测试人物被科力克近身后瞬间只剩下108HP..........
此测试说明，额外伤害取决于bb的基础生命值。比较符合“额外伤害数值等于死后的尸爆伤害”的结论。

测试六
将bb打至仅剩一点血，然后近身杀死。角色生命值接连减少两次，生命减少总值约为理论值的二倍。
重复数次，结论相同。
此测试说明，3bb死后的爆炸伤害会连续作用两次。

测试七
75%格挡，贴身杀死科力克，无损血，说明“3bb死时的尸爆不可格挡”纯属无稽之谈

测试八
修改鲜血荒地的僵尸攻击力为0，并赋予火焰强化属性，100FR/50DR状态去近身挨打，最大伤害为1.
此测试说明，普通金怪没有这个bug

测试九
1.11b版本重复测试二的步骤，数十次攻击，仅仅有普通攻击伤害，没有额外伤害的出现。
此测试说明，1.11b版本已经修复FE bug

结论已经很明显了
1. 具有火焰强化属性的3bb在近身时会有额外的伤害。
2. 此bug表现为伤害超高。此伤害近身才会有效，同攻击伤害分离，可以格挡。
3. 初步认为此伤害类似于光环的脉冲伤害，每两秒冲击一次，数值等同死后的尸爆伤害。
4. 此伤害会连续作用两次。
5. 多人难度下这个伤害不会增加。
6. 3bb死后尸爆伤害也会作用两次。
7. 1.11版本此bug修正。

那么，究竟在91D2暗黑战网，此bug的伤害是多少呢？
3bb的基础生命值为51296-64120. 按最坏的情况来看，尸爆伤害为64120*5%*50%=1603. 接连两次伤害，那就是3206. 如果是普通的vit角色，格挡率很低，按照75FR/10DR来计算，最终伤害就是3206/2*(1-75%)+3206/2*(1-10%)=1843
如果你中了伤害加深诅咒，DR-100，这个数值还要增加3206/2=1603
如果发动了临界一击，这个数值还要翻倍。

总之呢，战网上这个伤害没有那么可怕，但也不可掉以轻心，不小心的话即使没被秒也所剩无几。

最后给大家的建议:
1.硬撑生命的效果并不佳，可以看出，因为伤害是接连两次，再高的生命也经不起折腾。对付这个bug，还是75%格挡更佳，接连两次的伤害都格挡失败的几率非常小。
2.噩梦难度生命因子为35%，地狱难度为5%，也就是说噩梦的伤害更高，千万不要掉以轻心。
3.跑步模式block 25%，def=0，极其容易被击中，所以bh在瞄准过程中千万不要跑动，开封印前切换为走路模式。
4.分辨火焰强化的方法：记事本打开d2hackmap.cfg文件，找到“Fire Enchanted Desc:”这一行，后边字母改为"%1FFFFFFFFFFFFF"，这样游戏里看去，火焰强化怪物头上顶着一长串字母F，一目了然。
5.最安全的方法当然还是遇到该属性就开回城重新刷属性，尤其是对装备相对不好的新人来说。
