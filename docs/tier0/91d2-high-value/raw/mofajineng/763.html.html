5pp自动kb，ama冲天梯组合推荐

前提：pet都是刚毅、安头。

 
lsor：tp、清场，请完场后在5小队掉落的地点开门。普通战斗小弟。要清场快就无限。
bb：波波。噩梦进攻小弟戴丧钟。5x波。
bhpal：升20级祈祷，开净化光环。眼光小弟。个别情况帮忙清场。
smtpal：开狂热光环。拿元素武器。噩梦防御小弟戴丧钟。
ttama：冲天梯到99。tp、无限。雇普通防御小弟。
 
流程：lsor 下世界之石二层开门，ama下，bb下去波；lsor直奔毁灭王座清场，清场完在5小弟掉落地点开门，下bh、smt、bb；ama横推世界之石二三层、王座。ama只管扫怪，不用管4pp自动kb的，ama到99级时我的自动kb在都没挂过一次。在ama98级以后就不扫世界之石二三层了，直接扫chaos、kd、找exp神殿。
 
说明：bh、smt圣盾都不用开，bb也没有必要切换到副手，直接站神。自动kb有时候第二波小骷髅到处乱跑，或需要点时间。说是5pp自动kb，其实是4pp自动kb的。
