贫民格挡电法lsor配装选材与格挡配置指南
无限只要能RR也就是时间问题，贫民配置就是连无限都不需要投入就能玩，本帖的前提是“贫”，意思是不靠交易，也不使用小概率稀有装备，就能从白手起家搞定的普适型配置，购买电珠、格力风、等等都不属于本帖讨论范畴。

属性、抗性，这些与整体装备词缀有极大相关，各人游戏认知能力也不尽相同，所以每人都有自己的见解，适合自己的就是最佳选择。
比如抗性，并非只以75作为封顶考量，这对大多数玩家而言可能已经涉及到盲区了。

再说格挡，某种程度上可以说格挡型是法师的进阶，而VIT只是入门。
我每次都是VIT冰火法开荒，体力都点到300+了，血量很高，频繁掉血始终需要靠大量操作化解。
格挡作为一种稳定且高效的物理伤害过滤手段，主要作用是大幅减少物理攻击造成的掉血，最终表现为增加舒适度，减少操作压力。
这比VIT冗余的几百点血量有用得多，且那些血量还是虚的，根本就遭不住，所以格挡型装备贵是有道理的，这要看玩家的认知是否达到了这个门槛。
格挡型需要考虑的问题比VIT型要多得多，不配置格挡型人物是因为我的装备库还不足以完成一整套配置，随着装备逐步提升，能配起格挡型的时候我就会立即把小号武装起来干活。
之后VIT人物偶尔还是会用，只不过已经不作为我的一线主力了而已。

从VIT到格挡，这本身就是随着装备水平与游戏认知整体提升后的一种过渡，“贫民配置”只是格挡型的起步阶段，得先能玩起来，然后再逐步提升。
本帖适用于没玩过格挡型的新手，能够从属性、抗性、FCR、格挡、防御等全方位提供基础的参考与选择的思路。
练人物很简单，学会思考配置才是真的会玩了，我相信每个认真玩过的玩家都会逐渐形成自己的经验。

贫民格挡电法按需配装
1.新月+摩西盾，军帽，蛇皮，蛛网腰带，法拳，FCR黄戒指X2，项链马拉或其他，鞋子任意，板子补FHR不低于20。（4~6技能，减抗-35）
2.新月+精神盾，军帽，守护天使，蛛网腰带，法拳，FCR黄戒指X2/1，（乔丹之石X1），项链马拉或FCR≥10手工项链，鞋子任意，板子补FHR到60。（5~8技能，减抗-35）
3.新月+暴风盾，军帽，蛇皮，蛛网腰带，法拳，FCR黄戒指X2/1，（乔丹之石X1），FCR≥15手工项链，（马拉），鞋子任意，板子补FHR不低于20。（4~7技能，减抗-35）
4.塔拉夏套装+摩西盾/暴风盾，法拳，FCR黄戒指X1，乔丹之石X1，鞋子任意，板子补FHR到30。（7技能，减抗-15）

搞不起的话弄个其他职业垃圾ULC，高额属性和抗性也非常有用。
板子补抗性、生命/法力，词缀优先级：抗性>FHR>生命/法力
摩西盾配置抗性补满后，还有多余空格优先放生命+法力板子，再放电板或者MF板；
精神盾/暴风盾配置抗性压力大，可能没有多少空格了；
套装配置各项属性都非常充裕，包里随意。

既然是贫民，那肯定没有无限，PET用眼光即可，底材随意；
头盔补吸血，安达利尔的面貌/偷取骷髅/盗贼皇冠/吸血鬼的凝视，衣服穿都瑞尔的壳补抗+冰免，有形无形皆可。
镶嵌物根据武器底材攻速档位而定


降抗破电免
反手装备降抗聚气手杖，78级手杖能出现2级降抗（-37%能破106电抗），99级手杖能出现3级降抗（-41%能破107电抗）；
商店刷蓝色手杖时注意，手杖上FCR与聚气同属于后缀。

FCR
1.摩西盾配置，FCR=80，63档9F飞行，78档13F闪电输出
2.精神盾配置
 2-1.精神盾FCR变量28，FCR戒指X1，FCR=78，63档9F飞行，78档13F闪电输出
 2-2.精神盾+手工项链FCR=45，FCR戒指X2，FCR=105，105档8F飞行，78档13F闪电输出
3.暴风盾配置
 3-1.FCR戒指X1，FCR=80，63档9F飞行，78档13F闪电输出
 3-2.FCR戒指X2+FCR≥15手工项链，FCR=105，105档8F飞行，78档13F闪电输出
4.套装配置，FCR=80，63档9F飞行，78档13F闪电输出

抗性
3难度任务基础30，摩西盾63，蛇皮20~35，基础元素抗性=113~128，低于环境补抗性到75还需要62~47，如果项链和戒指没有多少抗性，军帽和蛇皮上也可以镶嵌#22+15全抗，如果连#22也没有就用#8补火炕、#9补电抗，剩下的靠戒指、项链、板子补齐，冰抗和毒抗50+即可。
精神盾配置抗性缺口最大，尤其是火抗，鞋子可以穿高抗黄鞋或者艾尔多绿鞋。
暴风盾配置相比精神盾配置稍微好一些，同样缺火抗。
套装配置抗性、生命、法力、MF都非常充足，套装属性还有25FHR。

防寒圆盾，力量需求77，防御力=95~110，任务打2孔
月精灵护盾，力量需求100，防御力=108~123，任务打2孔
洞穴巨魔巢穴骨盾，力量需求106，防御力=158~173，天生2孔或者非超强公式打2孔

格挡差异
摩西盾（圆型盾）力量需求=53，盾牌格挡=57%，90级时75%格挡需要敏捷=252
摩西盾（月精灵）力量需求=100，盾牌格挡=65%，90级时75%格挡需要敏捷=223
精神盾力量需求=156，搭配守护天使，盾牌格挡=62%，90级时75%格挡需要敏捷=233
暴风盾力量需求=126（156）/103（133），盾牌格挡=67%，90级时75%格挡需要敏捷=217
押韵盾（冷酷、月精灵、巢穴骨盾）力量需求=58、100、106，盾牌格挡=60%，90级时75%格挡需要敏捷=240
押韵盾（防寒圆盾）力量需求=77，盾牌格挡=62%，90级时75%格挡需要敏捷=233
摩西盾/精神盾FBR=30，暴风盾FBR=35，押韵盾FBR=40，都符合格挡需求；
注意精神盾配置不要换穿别的衣服，FBR=0时格挡动作会对施法带来极大干扰。
替换押韵盾时要注意敏捷需求对格挡率的改变。

配置解析
新月提供了非常可观的-35电抗，还有2EK，成本超低（#13+#22+#3），贫民电法武器首选。
摩西盾配置，元素抗性全面且充足，力量基本无需求，整套配置成本最低；
精神盾+守护天使配置，全技能高1级，法力和FHR都更高，敏捷需求略低，力量需求大，抗性缺口更大，优点是未被降抗时全抗有90。
暴风盾配置，整体力量需求比精神盾配置更低，盾牌镶嵌减需珠后力量只需要103就可使用，敏捷需求最低，元素抗性缺口也不小，物理抗压能力强。
套装配置，拥有最充裕的技能、抗性、生命、法力、还有168MF和25FHR，可以说非常全面了，唯一不足就是减抗略低，有5个孔可以镶嵌电珠来补，珠子减抗优先，增伤无所谓。
套装摩西盾配置如果想调整力量，漆甲力量需求84，高于摩西祝福之环（圆型盾）力量需求53，漆甲可以选择镶嵌减需珠进一步降低力量需求为52；或者升级摩西祝福之环（月精灵护盾）力量需求100，然后盾上镶嵌一颗减需珠降低力量需求为85来匹配漆甲。
套装暴风盾配置物理抗压能力更强，元素抗略低，力量需求最低103高于漆甲，少1孔。

 古龙皮，防御力=800~1034，力量需求=84
如有降力量需求，可以镶嵌-15%减需珠，力量需求=72
2.守护天使
 圣堂武士外袍，防御力=770~825，力量需求=118
 地狱锻甲，防御力=（1178~1263）~（1484~1590），力量需求=196
守护天使配置如果整体抗性充足，可以镶嵌-15%减需珠，力量需求=167
3.摩西祝福之环
 圆型盾，防御力=156~179，力量需求=53
 月精灵护盾，防御力=（302~345）~（344~393），力量需求=100
如有低力量需求，减需珠X1，力量需求=85；减需珠X2，力量需求=70

格挡型人物提升基础防御力被高级大叫增益后的收益很大，尤其是搭配反抗光环PET，再加上自身冰甲，防御力可以轻松上9~10倍，2W防+75%格挡对物理攻击的过滤能力非常优异。

格挡与打击恢复这两种硬直动作期间都能正常进行防御和格挡检定，格挡型比非格挡型拥有更高的综合防御率，所以FHR的需求相对更低一些。

格挡上限75%，必然会有攻击漏过格挡检定，所以血少防低人物即便满格挡+高FBR，也还是有不小的概率会被HR打断。
玩家首先应该做的是提高综合防御率、血量、抗性，尽可能减少HR触发率，然后再适当提高FHR减少人物被连续触发硬直动作留在原地的可能性，格挡型由于稳定高效的伤害过滤能力，在做好前三项之后，FHR就可以不用堆很高了，具体要多少得看前三项综合强度。
