镰刀无限法师life上限=mana上限的加点计算
B=[(1+y)*(a+2n)-b*(1+y+x)+c+d-e]/[2*(2+2y+x)]

A=n-B

注：计算完成B为小数，向下取整数，略掉小数部分，进位会导致bo后mana上限高于life上限。

参数：
n：加完力量剩余属性点总和。
A：活力应加属性点。
B：精力应加属性点。

a：包括1级时获得的life，升级时获得的life 奖励，act3 quest1的20life任务奖励。装备和背包里charm加的life总和。
b：包括1级时获得的mana，升级时获得的mana奖励，装备和背包里charm加的mana总和。
c：装备上的+life base char's lvl(根据级别+life)，无限上加的是vit,要乘以2换算成life
d：装备和charm上+活力（包括+所有属性）之和.同样乘2换算成life，全属性的可以不管。
e：装备和charm上+精力（包括+所有属性）之和.后边同上
x ：soj和纱织的+mana xx%的属性之和（例如soj加25%mana,身上带了两只则x=0.25x2=0.5）
y：各种+max life%的技能（如bo，dru的变身，招灵等）和装备（如tp+的%max life）总和（例如如bo+65%,则y=0.65）

例子：单机一个0STR的90级镰刀sor

装备：
镰刀WX
3电/20fcr镶 9STR/15猪*2
2skill/17fcr/85mana
GLP 镶 -15req/15R
法拳
技能腰带
火电毒30的3抗鞋（力量需求≤74）
SOJ一只  10fcr/16str/40life
背包：极品ULC  10/15usc  40life lgc*6  20/17 sc*5 回城书一本

加点：
力量=74-初始10-珠子9*2-ULC20-USC10+戒指16=0
敏捷不加
活力先等等
精力先算

n=升级奖励445+ACT3奖励15=460
a=初始40+ACT3奖励60+升级奖励89+装备背包一共380=569
b=初始35+升级奖励89*2+装备背包一共190=403
c=无限45vit*2=90
d=0
e=0
x=soj0.25+腰带0.05=0.3
y=bo技能200%加成=2

套入公式B=[(1+y)*(a+2n)-b*(1+y+x)+c+d-e]/[2*(2+2y+x)]

B=[3*(569+460*2)-403*(1+2+0.3)+90]/[2*(2+2*2+0.3)]

B≈256.11=256

A=460-256=204

活力=初始10点+204=214点
精力=初始35点+256=291点

生命上限=(a+2A)*(1+y)+c+d
法力上限=(b+2B)*(1+x+y)+e

生命上限=(569+204*2)*3+90+30*2=3081
法力上限=(403+256*2)*3.3+30*2==3079.5≈3079.5系统向下取整数=3079
