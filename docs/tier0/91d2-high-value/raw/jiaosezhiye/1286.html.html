拳击BUG近战野蛮人玩法省心攻略
近战野蛮人的主要攻击手段有4个：
旋风、狂乱、狂战士、专心

劣势：
专心，既没有旋风的超快攻速，也没有狂乱的攻速移速，更没有狂战士的全魔法伤害类型优势。

优势：
（1）专心的主要优势是动作不会被 格挡/打击恢复打断，也不会被击退/眩晕，在这一点上与旋风和狂乱相同，同时在挥动武器的动作过程中有大量防御ED%加成。
（2）相比同样是主手单次攻击动作，狂战士不仅攻击时防御清零，而且还会被打断，所以狂战士无论是否带盾，都得把小怪吼开，万军丛中只取上将首级。
（3）专心不被打断，意味着持盾使用时就有了和旋风、狂乱一样的持续输出能力，且凭借更为优异的防御%，能在乱军之中一个一个从容地把小怪全部砍死，物理抗压能力比旋风和狂乱更强，更不会像狂战士被围攻时那样频繁掉血，所以专心最适合用于持盾清场。
（4）固定2点法力消耗也是最低，对LM需求极低，有就行。

技能加点（共107）：
战斗：重击20、击晕1、专心20、狂战士20
呐喊：狂嗥1、大叫1、战斗体制20
专家：剑系掌握20、耐力1、加速1、铁布衫1、自然抵抗1

伤害：
（1）专心在伤害类型上与狂乱类似，有狂战士协同加成，每一级狂战士都能增加1%魔法伤害转化，最高20级=物理伤害20%转成魔法伤害，也使专心有了和狂乱一样的被动破物免伤害能力，且不会过于影响双吸。
（2）近战伤害在悔恨加持下一般都挺强，再加上DS、CB、20%魔法转伤和强化BUFF，已经很全面了，物理魔法火焰，所有怪都打得动。
       CS=21%+、DS=83%+，总双倍机率=21+63*（1-0.21）=86%+
       CB=35%+15%=50%
（3）技能命中加成仅次于狂战士，比狂乱和旋风都要高，当然了在专业强化BUFF加持下命中都不是问题。
（4）技能伤害加成仅高于旋风，比狂乱和狂战士都要低，确实是4个技能中综合伤害最低的，伤害加成不是专心的重点。
（5）专心伤害其实并不低，只是比狂乱少了一些技能ED而已，只要能用上悔恨，输出就不是什么大问题。
（6）没人玩专心主要是不用拳击BUG时攻速不够，伤害也不如另外三种有亮点，玩的确实有点憋屈，还好有解决方案。

协同路线：
由于专心的协同加成与狂战士有较大区别，所以当作为狂战士BD的副输出技能时无法达到全效。
应该说除了旋风以外，另外3种主攻技能，在主要路线上的协同技能都有很大区别。

拳击攻速：
由于专心和狂战士都是套用普攻动作，所以都能利用拳击BUG来突破9F武器攻速极限，进一步提升攻速档位。
选择拳击BUG时，得按常用武器底材需求来使用不同的BUG属性。
既然戴盾那肯定得加满格挡，90级野蛮人用暴风盾75%格挡需要敏捷=203，幻化敏捷需求136首先PASS，换力量。
力量需求：
瑞韩德125，瑞韩德也是最常用的悔恨底材之一，做了不亏；
狂战斧138，这个就很少有人用了，因为耐久26只有剑的一半实在太低了，修得心烦；
巨神刃189，这把剑也很少有人做，因为旋风不用拳击BUG，必须做出39ias才能达到极速4F，成本不低，除非有大佬就喜欢玩这个，可以试试看能不能有缘搞一把便宜的不达标残次品；
巨神剑182，这把剑没人用来做悔恨，因为旋风不可能极速，以通用性考量一般不用。

拳击BUG会使武器上的IAS失效，套用空手动作会忽略武器基础速度，空手动作忽略武器攻击范围差异。
所以玩拳击时，WSM、WIAS、range可以全部无视。
瑞韩德、狂战斧、巨神刃，接下来从这三把武器中挑选。

拳击BUG想用悔恨+盾牌组合，一般使用暴风盾，in减需珠后暴风盾拳击BUG触发区间=103~125
面板力量=103，暴风盾切换形成拳击BUG后，实际力量=（103~125）+30=133~155
在面板力量=103时，不超过155力量需求的武器只剩下瑞韩德和狂战斧符合使用标准，综合考虑最后还是推荐用瑞韩德。


关于敏捷
同时具有力敏双属性需求的装备，如果只满足其中一种属性，还是不能装备，只要挑选其中一种卡就行了。
瑞韩德之剑，力量需求125，敏捷需求94
只要控制面板力量在103~124，瑞韩德就能随便卡拳击，敏捷只要面板≥94即可，往上加多少敏捷都不影响。


瑞韩德悔恨、减需暴风盾，单机添加抵抗火焰光环和荆棘光环，用于观察。
切换后，双光环都生效，说明剑盾均已形成拳击BUG，敏捷早就超过瑞韩德需求了。
不用幻化是因为力量需求25太低了，实在是没办法，不然肯定用幻化了，还不用修理。


PET配置：
用拳击BUG玩专心时，身上其他部位IAS总量要根据PET配置而定。

1.丧钟米山
米山是一位刚硬猛男，他能与你并肩作战，衰老减速减伤害减物抗攻防一体。
IAS以不低于48为标准
48=8F、86=7F，中衰老或12级圣冰时，攻速档位还会下降。

具体选择48档还是86档就要看头部装备了。
48档：安置手+20、大君+20、吉永inIAS+15，总IAS=55
非减速时8F，被单减速时13F
86档：安置手+20、大君+20、吸血圣王+10、三孔头inIAS+15*3，总IAS=95
非减速时7F，被单减速时11F

被双减速叠加时，48档只有32F，86档只有22F，注意不要硬抗。

2.信心京子
京子是一位娇柔妹子，生存能力堪忧，你必须时时刻刻保护好她。

注意12级狂热IAS要求65，13~15级IAS要求60
非减速6F极速：
安置手+20、大君+20、吸血圣王+10、吉永inIAS+15，总IAS=65


单减速：
IAS65=9F、IAS95=8F


双减速：
比没狂热时好很多了，几乎减半，可见狂热对衰老和圣冰的攻速对冲很有效。

其余配置：
近战野蛮人的其他部位装备主要以IAS为主进行选择。
胸甲刚毅，戒指乌鸦+双吸，鞋子蚀肉骑士，基本都固定。
板子同其他近战BD相同，以尖锐+生命为主。
抗性一定要补满，有能力的溢出。
高防御+高格挡BD，FHR不重要，有吉永的30%足够了。

关于BO：
专心BD加满了战斗体制能自己BO，但是大叫太低了，如此注重防御的BD就显得有点儿尴尬了。
最好还是让专业BO号来加BUFF吧，不仅效果更强，持续时间还长。

关于3孔头：
选择擎天神等力量后缀时注意带上头后，面板力量控制在103~124区间内。
表现、伤残、八目鳗都行，具体看财力。
永生头其实很不错，配合暴风盾，物理抗压更进一步。
嫌跑得慢也可以用速度头，30%跑速能加快不少。

总结：
总的来说，专心是近战野蛮人攻守最平衡的BD。
悔恨+暴风盾，同时使用拳击BUG，应该是专心的最佳解决方案了。
并不是说非拳击不可，只不过专心确实在攻速上比旋风和狂乱差一大截，再不利用拳击，输出就差太远了。
在怪堆里砍怪，专心就是省心。
