闪电火墙双修法师，速刷bug超市完全攻略
五开： bb + 闪电火墙法 + 200fcrEs新星法 + 狂热pal + 审判pal。

一、流程

200fcr法师直奔大翅膀，开门进人bobo。然后电法tp杀西希，混沌大臣，tp看怪到超市门口，此时站神们已经解决大翅膀。 200fcr开大翅膀旁边最后一个封印，tp捡东西到超市门口，一局结束。全流程算上找凯恩和捡宝石能轻松控制在1分钟之内。

二、人物选择

考虑到全局要有多次施法，施法速度最快的就是法师和bb，施法提高一档，一局节省两秒。这是其他地方的优化很难达到的提升。所以tp和捡东西的，肯定是200fcr法师（比bb好是因为心灵传动操作更方便，而且法师可以用旭日从而更安全）。

负责杀西希和大臣的，闪电火墙平均每局控制在3-4秒（1/6概率电免西希3-4秒，非电免西希和混沌都是1-2秒），其他角色很难有这个效率。圣骑虽然击杀很快，但是tp差一档，没有心灵传动，这块要额外耗时两三秒，很难赶上法师效率。

站神除了bb，选了狂热pal和审判pal。审判不光降低抗性以增加200fcr的清场输出（站神审判可以-150抗性，是无限不可替代的），还可以大量降低怪物防御，从而不用考虑pet的命中率（这方面比祝福瞄准好太多）。如果六开可以增加一个专注pal。两个pal战神堆到300mf+。狂热用悔恨准备补刀。审判用愤青可以在进门的时候击退厄运骑士，也可以给大菠萝上元素bug。

三、站神解决大翅膀

200fcr法师直接站到大翅膀堆里右键新星移到其他窗口。新星不光可以击杀非电免灵魂，即使电免，新星也能迅速清场，让pet集中输出灵魂。电法杀另外2boss的时间足够200fcr新星和4小弟清理掉大翅膀。4pet：A5(新月)负责aoe闪电输出  A5(击头+刚毅)负责物理输出  A2(丧钟+力量)负责上衰老  A2(眼光+圣冰)配合TP。狂热pal左键白热手持悔恨流亡辅修神圣冲击，随时准备补刀（新星法选位好的话，双热骑几十把轮不到出手机会）。

补充一下 如果三开，那么200fcr要修暴风雪，namelock灵魂之后鼠标移开。这里的麻烦点是灵魂有三成概率冰免，要用电法给两下子再去杀另外两boss，用旭日站神安全性很好，bb要拿一个愤青。

四、200fcrTpSor的技能和装备

不合宝石的话，200fcr只带ulc和2小符。6*回城书放盒子里，盒子放箱子里。200fcr法师的物理防御只能靠ES，这样开封印放灵魂的时候可以多抗几刀，实操下来很安逸，不会挂掉也不用喝紫瓶。（不需要cta，法师切换武器bo两下也很麻烦很耗时，没必要）（max新星，支配电，心灵传动，能量护盾）

五、全员用旭日

除了bb用3bo项链+吸火手套，其他包括200fcr法师在内都用旭日。不光是为了防止火焰伤害，也是为了保持血量健康。旭日吸火效率最高的是-25火炕，大翅膀喷一口回200，如果是0或者-50，喷一口回160。（吸火手套吸火效率最高是+75火炕，喷一口回40，既对火炕提出很高的要求，又回血太少，不推荐。）

六、核心是提高电法的防御

电法用旭日意味着不需要因为血量喝紫瓶，随时满血。没有旭日的话，半血的时候要不要喝紫瓶，都很麻烦。不喝的话容易翻车，喝的话捡一个紫瓶浪费大半秒还费心费神。对效率影响最大的是防御。防御够高，才可以大大方方的站到西希面前，出手走人。输出终归就是三四秒，提升2技能也就是节省0.3秒。但是因为站不住tp一次，0.3秒没了，捡一个紫瓶喝，1秒没了。绕着西希盘旋两圈找落地位置，2秒没了。

紫瓶基本都是被西希法力燃烧的情况下才喝，西希小弟对防御33k的电法的命中率在13%，站位合适的话基本可以不被击中，实际操作下来每一百把消耗十来个紫瓶。 混沌的法力燃烧速度很慢，bo后的法力完全够用。

应对非电免的西希很轻松，一是闪电只需要一两秒解决战斗，二是闪电能把菜刀队打僵直，不会被摸到。如果电法挂掉只能是伤害加深面对闪电无效西希。火墙烧西希的时候，菜刀队围过来，要站住三秒才行。同时闪电强化西希小弟攻击附带闪电伤害，所以提高火墙伤害、防御、生命、fhr、闪电抗性都非常重要。

七、要不要MF

mf对产出的提升只对击杀3boss有效，本赛季做了详细的计算，因为3boss的染色率高，光看暗金的产出都值得提高mf，计算下来电法150mf提高西希混沌2boss的暗金产出占全局的0.7%+，相当于节约60秒*0.7%+=0.42+秒，提高2技能可以节约4秒闪电时间的10%也就是0.4秒，这么看150mf是大于2技能的。何况这里只计算了暗金的提升没有计算黄金的提升。但是超过200mf之后的边际提升下降严重（3boss的戒指黄金概率在200mf到顶点），就不太值得硬堆。所以200mf以下认为75mf优于1闪电技能。当然这些都是0.几秒的差异，对整体的60秒影响不大。整体还是流程设计、防御带来的流畅度提升最重要。

八、闪电火墙法师的装备

电法：脾气 权冠 技能腰带 2*soj 法拳 旭日 ethIRE 物免盾 沙暴。身上都是lgc和电抗小符。105fcr，60fhr，火炕-40，电抗75+，毒抗30，bo后2400+血，1000法，120mf，便宜实用。
技能是20火墙1火焰支配1冰甲，其他闪电相关。2道火墙烧死西希是流畅程度的重要指标，火焰伤害不够的话可以用几个火板或者多加一些火焰支配。
反抗小弟无限，头和甲尝试了多种方案，现在用sa刚毅和吸血抗性力量黄金头环，感觉最硬，击杀火电双免（1/66概率）的混沌也最轻松。

可以讨论的是甲和盾。防御肯定要够高才能抗住西希小弟。如果用电盾，甲就只能用慎重。如果用物免盾，可以用防御稍微弱一点的甲，比如ethIRE，阿凯尼，tp+，暗金海妖壳（这个防御过关，但是伤害比前三个低）。其他如魁黑刚、荣耀之链、古龙皮、血甲、奥玛斯都防御不过关。

电法的输出，由于混沌大臣抗性高，-5电珠提升略大于1闪电技能，计算产出大致按1技能=1电珠=75mf。
下边计算的技能输出是伤害高低，综合输出是考虑MF的单位时间的产出。防御分数是计算命中和格挡之后能抗狂热+伤害加深的西希小弟几次攻击。

ethIRE高分是因为2100的防御对安全性的提升非常高。输出方面1技能1孔120mf也很优于其他。

如果不考虑mf的话，可以用阿凯尼，阿凯尼技能高一级，防御比ethIRE差一些，是激进派的选择。

实际手感还是用ethIRE更从容，除了法力燃烧啥都不怕。上赛季尝试过一阵子阿凯尼，站在怪堆里不太踏实，紫瓶消耗多一些。其他搭配都不值得了。
