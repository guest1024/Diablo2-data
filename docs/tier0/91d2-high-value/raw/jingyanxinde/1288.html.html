低PP通用型格挡顶球新星电法lsor（牛场+地穴+古代通道）
“贫民格挡电法”技能改为新星主攻，再配合顶球，用高防御+高格挡来过滤伤害，用护盾缓冲伤害，眼光续航。
依托新星8码半径不限目标数量的强大AOE能力，1-3PP低强度场景轻松割草。
同样不需要无限和格里芬也能刷牛场、地穴、古代通道，伤害、防御、续航都足够，自己MF基本都能入手。

骨牢聚气修理比较贵，基德和边缘要备好，刷牛场前有打钱号的打点儿钱，没有的平时MF卖垃圾攒个一千万能修很多次。
如果不想聚牛也行，直接进去一群一群杀，对法力要求高一些。

我一般不推荐格挡+顶球，因为这种BD一般只用来割草低强度场景，天生的机制冲突使之缺乏升级兼容性。
用无限反而会使整体失衡出现续航缺陷，进一步影响防御维度，所以也没必要花大价格投入电肉板，日用装即可。


加点（90级法师总计101点）：
电系97点：新星20、闪电支配20、心灵传动20，闪电20，静态立场13，闪电弹1，连锁闪电1，雷云风暴1，传送1
冰火4点：冰甲1，霜之新星1，冰冷支配1（加减速时间），温暖1

配置：
全身装备带火炬+10技能，BO+1，11级全技能，7块电光板。
建议用新月暴风盾配置，军帽和蛇皮各镶嵌一颗电珠，以-5为主，+多少随意，减电抗可以达到-45。
暴风盾镶嵌减需珠形成拳击BUG，只要103力量即可切换生效，90级时75%格挡需要敏捷=217，剩余点数加体力。
反手装备9蛋杖，带不带冰甲无所谓，带冰甲就可以省1点技能。
鞋子用≥15力量骨髓行走，用其他物品提高力量正常BUG穿戴，记得最后再穿戴一次鞋子，使BUG永久生效。
骨牢聚气13次，修理费用353792，带40%基德和15%边缘弓后，维修费用降至251193。

施法（FCR105）：
无格里芬，两枚FCR金戒指，高法力为主、力敏抗皆可；项链FCR≥15，高法力为主、力敏抗皆可。
有格里芬，项链可以不用搞手工的，戒指还能用一枚乔丹，穷人一般不考虑格里芬。

防御：
在牛场中绝大多数伤害都来源于近战物理，所以高防御+高格挡有天然优势。
蛇皮一定要升级为古龙皮，抗性不重要，防御值要上1K
暴风盾140、军帽140、古龙皮1K、蛛网腰带130、扩展级法拳60、骨髓行走200
装备防御值可以达到1670，暴风盾90级增加防御值=90*3.75=337
法师敏捷217，Base_Defence=[217/4]=54
人物总防御值可达2K左右
