天堂基础盾击丁3%残血阶段盾击与热诚DPS分析
先说结论：

3%残血阶段内，劳模和莉莉丝用热诚能更快击杀，但必须要有高级强化BUFF加持才行，且都需要开审判光环。

开审判斩杀莉莉丝，是不是反直觉了？
因为热诚凭借高元素附加和审判减抗，使综合伤害做到了在"一定程度上"接近悔恨盾击。

只是热诚要计算命中和格挡，所以伤害并不稳定，且天堂相比悔恨进阶的主要劣势是盾击和热诚的物理伤害太低，吸不了多少血，可能会导致血线不稳，需要时刻关注血条，不要省紫瓶。

3%之前还是推荐使用盾击压血，必中又快又稳定效率高，3%之后由于CB伤害大幅降低，元素伤害占比开始大幅超越CB。
劳模全程开审判，最后3%，盾击3.36秒，高级强化热诚2.32秒左右。
莉莉丝3%之前用狂热盾击，最后3%，狂热盾击5.04秒，3%之后换审判热诚，高级强化审判热诚2.32秒左右。

2.32秒计算综合命中率后大约就是2次5/5/9F的热诚5连击，这只是一个用来参考的平均值，实战中会受到命中率加格挡率的影响而产生波动，如果运气好的话可能一个热诚就秒了，运气不好可能要3次或以上......，总的来说确实能更快一点击杀(具体伤害和攻击序列请看下文)

如果有准备专业强化法师，有兴趣的可尝试一下审判热诚的斩杀方式。

使用的时机要判断准确，如果边上已经围了一圈小怪，就别用热诚雨露均沾了，直接盾击锁定BOSS。

注意，请勿对大菠萝(110)和巴尔(115)两兄弟使用此法，火抗太高了，强化对热诚的火伤增益巨幅下降。
还好只有劳模和莉莉丝比较难，另外四个白菜~



----------------------------------------------------------------

牺牲只提供给热诚伤害ED协同，加热诚同时加命中和伤害，由于剩余点数很少，区别不大。

审判大幅降低BOSS防御值，对命中有不可忽视的影响；天堂之光的-33%目标防御对BOSS有一半效果，对命中率有微小提升；
虽然高级强化BUFF对攻击准确率也有很大提升，但由于审判和天堂已经把目标防御值降到非常低，所以对最终命中率几乎没有多少影响了，主要意义在于火伤对热诚的增益巨大。开狂热不开审判时，强化的命中对热诚还是还是很重要的。

由于PLD格挡恢复动作极短，所以本数据推演并未将格挡动作带来的影响纳入攻击时长计算中，望知悉。
