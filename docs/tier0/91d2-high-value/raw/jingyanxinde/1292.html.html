“杰海因消失之谜”与“水井KEY”的神秘关系
鉴于总是有小伙伴搞不明白为什么有时杰海因会消失，又时又不会，这种灵异现象，还是掰开来仔细讲讲。

水井KEY制作流程：
bus杀安姐，角色a留在act1，角色b去act2，b取仗+符，合成后与卓格兰说话，目睹召唤者死亡，再与城内npc说话，bus飞到插仗处开门等待，角色a从瓦瑞夫到act2，角色b插仗后角色a通过传送站回act1，至此水井key-a制作完毕。

1.流程梳理：
角色a不在第一时间从瓦瑞夫去到ACT2，而是等待角色b在插杖之前才去，这样合成杖任务和击杀召唤者任务在角色b完成时角色a就都会错过，当杖插下去的时候，角色a和b同时完成了插杖任务（赫拉迪克之杖），此时角色a直接从小站回ACT1，途中不与任何一位NPC有交互，这样角色a就等于跳过了主线任务链，在尚未触发杰海因和卓格兰任务时就拥有了古墓开门的能力。

2.为什么要跳过任务？
因为水井KEY的主要作用是让杰海因留在瓦瑞夫身边迎接来宾，如果主线后续任务已经触发，杰海因就会离开水井，所以要用跳过中间步骤的方式来实现古墓开门的同时不触发主线任务链。

3.角色任务状态与游戏房间状态
建立游戏房间的角色决定了该场游戏的任务起始状态，如果是水井KEY-a建立的游戏，那么该场游戏ACT2的任务状态就会处于只完成了第二个插杖任务，而三（堕落的太阳）、四（神秘避难所）、五（召唤者），这三个任务都为未完成状态，此时杰海因就会在水井边等待，如果是正常完成主线任务的KEY角色建立游戏，杰海因就会在王宫等待。

4.杰海因消失之谜 与 bus使用注意事项
（4.1）水井KEY建立游戏后，bus不要在ACT2下线的状态下直接进入游戏，这样一进游戏bus就处于ACT2场景内（水井边）且靠近王宫范围，此时只要bus一出城，就会导致杰海因立即消失（水井与王宫两边都不在）；
（4.2）只要bus不靠近王宫触发范围就不会使杰海因消失，比如与ACT3马席夫对话回到ACT2，此时直接再从马席夫去ACT3，杰海因并不会消失；注意：ACT2小站、城门、下水道、王宫入口、回程卷传送门都可触发。
（4.3）水井旁边的杰海因需要小号守一下，原因是当击杀督瑞尔事件触发时，城里在没人的情况下杰海因会随游戏房间的任务状态更新至王宫处；当BUS击杀督瑞尔时，小号在水井边上看着杰海因，他的进度就被小号控住了，此时BUS击杀督瑞尔，杰海因就不会去王宫了。
（4.4）bus在与泰瑞尔对话后，要依次回城与杰海因和马席夫对话，重置bus自身的任务链状态，不然下次游戏时bus再与泰瑞尔对话将无法触发任务链，导致小号无法完成任务。
（4.5）未完成插杖任务的新角色建立的游戏，当bus靠近王宫或从水井边进入ACT2也会触发杰海因任务链状态刷新，在bus出城后杰海因会立即从水井消失而出现在王宫门前。

5.水井KEY游戏内的杰海因消失实验
（5.1）城内角色使用以下方式离开ACT2城镇杰海因立即消失：
两个方向的城门、南北两个下水道入口、王宫入口、小站、传送门。
（5.2）城内角色使用以下方式离开ACT2城镇杰海因不会消失：
与马席夫对话从ACT3到ACT2后立即与马席夫对话再回到ACT3、直接退出游戏。
结论：
一.玩家人物无论有没有任务进度，只要进入ACT2绝大部分区域，一出城就会使杰海因从水井边回到王宫，而水井KEY进度跳过了主线，杰海因既不能在王宫，又已离开水井，所以消失了。
二.杰海因消失了没关系，只是暂时的，重新开一场游戏他就又回来了。

6.正常KEY
与水井KEY的主要区别就是，正常KEY的主线任务链触发状态与bus进度一致，不会导致杰海因位置冲突而消失，他始终都会在王宫等待任务交付，bus同样需要重置自身的任务链状态，正常KEY建立的游戏可从王宫下去开门快速穿梭ACT2城内区域。

7.水井KEY失效
水井KEY失效是由于KEY号与杰海因有了交互，导致任务链被触发，虽然合成杖与击杀召唤者的任务都没完成，但是主线任务插杖已经完成了，所以杰海因的任务进度被强制更新到了王宫，以后该KEY号再建立的游戏，杰海因不会再消失了，因为他会直接从王宫出现，等于“水井”失效了。没有任何补救办法，只能重新做一个水井KEY。

8.水井KEY失效实验
（8.1）水井KEY号与NPC交互
我在IMPK战网用5个RR号在噩梦做了一批实验，5个水井KEY在插杖前全部从瓦瑞夫进入ACT2，插杖后，第1个KEY号直接走小站回ACT1，第2个从瓦瑞夫回ACT1，第3-5个分别与杰海因、卓格南、亚特玛对话，然后再走小站回ACT1，期间全部组队在一起。
最后结果：
只有与杰海因对话的KEY号“水井”失效，杰海因挪到了王宫，其余KEY号的杰海因"水井"功能均正常。
说明水井KEY只要不与杰海因对话，就不会更新杰海因的任务进度，走小站或者瓦瑞夫回ACT1也都不影响，即使组队都不会影响角色各自的任务进度。
（8.2）水井KEY在游戏内时其他号与NPC交互（水井KEY全程不与他人组队）
（8.2.1）水井KEY建立游戏后没有退出房间，BUS击杀督瑞尔，全部退出，水井KEY未失效。
（8.2.2）水井KEY建立游戏后没有退出房间，BUS击杀督瑞尔后与泰瑞尔对话，全部退出，水井KEY未失效。
（8.2.3）水井KEY建立游戏后没有退出房间，BUS击杀督瑞尔后与泰瑞尔对话，小号任务进度为已经与泰瑞尔对话过的状态，任务进度为与杰海因对话，小号只与杰海因对话，全部退出，水井KEY未失效。
（8.2.4）水井KEY建立游戏后没有退出房间，BUS击杀督瑞尔后与泰瑞尔对话，小号依次与杰海因和马席夫对话，全部退出，水井KEY未失效。
（8.2.5）水井KEY建立游戏后没有退出房间，BUS击杀督瑞尔后依次与泰瑞尔、杰海因和马席夫对话，小号任务进度为与杰海因对话但是不对话，全部退出，水井KEY未失效。
（8.2.6）水井KEY建立游戏后没有退出房间，BUS击杀督瑞尔后与泰瑞尔对话，小号和BUS都依次与杰海因和马席夫对话，全部退出，水井KEY失效。
说明当水井KEY在游戏房间内时，哪怕是在ACT1，并且没有与其他任何人组队，依然会被游戏房间的任务进度所影响，当其他角色在ACT2城内交付任务时，只要有一人（包括BUS）不做完全套交接，水井KEY就不会失效，如果城内所有人都完成了任务并离开ACT2，水井KEY的进度就会被房间进度更新导致失效，失效后杰海因就会出现在王宫而不是水井。
结论：
一.水井KEY任何时候都不要与杰海因对话
二.水井KEY建立游戏，等其他人物进入游戏后，水井KEY就立即退出，永远不会“被失效”

9.正常KEY杰海因也会玩消失？
有时批量做完正常KEY后，明明主线任务已经做到最后一步了，用KEY建立游戏会遇到一开始杰海因就不在，这就很奇怪了。

用该KEY退出游戏，重新建立游戏，杰海因还是不在！
别慌，用其他角色新建游戏，该KEY加入游戏后与杰海因对话即可，然后用该KEY新建游戏，杰海因已经出现在王宫了。
有可能是正常KEY在进入ACT2后未与杰海因对话，他生气了~哈哈~
并非是不可逆的，解决办法也很简单，具体原因暂不追究了。
